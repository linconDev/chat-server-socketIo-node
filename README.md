# Chat Server Socketio

## Description

Websockets server, developed for exchanging messages in a chat

## Requirements

- NodeJs v20.14.0
- NPM or Yarn (We recommend using yarn)
- TypeScript, ts-node and nodemon ( ``npm install -g typescript ts-node nodemon`` )
